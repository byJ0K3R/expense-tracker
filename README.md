# 💸 Учёт расходов

Android-приложение для учёта личных расходов. Написано на Python + Kivy.

## Возможности

- ✅ Добавление расходов (сумма, категория, дата, комментарий)
- 📋 Список всех расходов с удалением
- 📊 Статистика за месяц (итого, по категориям, средний расход в день)
- 💡 Рекомендации по экономии
- 📤 Экспорт в CSV
- 💾 SQLite — данные хранятся офлайн

## Сборка APK

APK собирается автоматически через GitHub Actions при каждом push в ветку `main`.

1. Загрузи код в репозиторий GitHub
2. Перейди во вкладку **Actions**
3. Дождись завершения сборки (~25-40 минут)
4. Скачай APK из **Artifacts**
